Welcome to TeXample.net. There is not much to see here yet. In due time this site will contain miscellaneous TeX-related tools and resources. Until then, feel free to visit some of my existing TeX-related projects.

For more information visit my main web site www.fauskes.net.

Resource Description
PGF and TikZ examples gallery The purpose of this gallery of examples is to showcase PGF and TikZ, and to be a useful learning resource.
BibConverter BibConverter is a simple web application for direct conversion of data from online bibliographic databases to the BibTeX format. It extracts data from text copied directly from the web page and outputs BibTeX records
Dot2TeX A tool for converting graphs generated by Graphviz to formats suitable for use with LaTeX.
Dot2TeX(i) The dot2tex(i) package allows you to embed graphs in the DOT graph description language in your LaTeX documents. The dot2tex tool is used to invoke Graphviz for graph layout, and to transform the output from Graphviz to LaTeX code. The process is automated if shell escape is enabled.
A Blender to Sketch exporter A Python script for exporting Blender objects to the Sketch scene description language.
A Blender to TikZ exporter A Python script for exporting Blender curves to TikZ format for use with (La)TeX.
An introduction to Sketch 3D An introduction to Sketch 3D, a tool for creating simple 3D scenes for use with (La)TeX.
Tips and tricks for LaTeX and friends A collection of accumulated tips and tricks. Slightly out of date.
Finding the right bibliographic/reference tool A review of bibliographic tools. Slightly out of date. Read the user comments for the full picture.
Embedding equations in HTML A Python script for rendering LaTeX equations embedded in a HTML document.
